CSS offers 2-D and 3-D Transformations

2d transformations are

translate/move, rotate, skew, and scale
  1. transform : translate(X-axis translation value, [Y-axis translation value]) | translateX() | translateY();
  2. transform : rotate(angle); - clockwise-positive angle otherwise, anticlockwise
  3. transform : scale( width [, height] ); - width and height may be in px or percentage - scaleup positive value otherwise, scaledown | scaleX() | scaleY()
  4. skew( [, ] )
  5. transform: matrix(a, b, c, d, e, f);
